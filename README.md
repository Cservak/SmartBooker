# SmartBooker

SmartBooker is a modern full-stack application that allows teachers to easily book rooms for their classes, all through a visual interface.

- [Installation](#installation)

## Features
- User registration and login
- Room listing and booking
- Admin panel for managing bookings
- Responsive user interface

---

## Technologies
**Frontend:**
- React 18
- Vite
- Tailwind CSS

**Backend:**
- Java 17
- Spring Boot
- Maven
- H2/SQL database (for development)

**Other:**
- Git version control
- SSH or HTTPS connection to GitHub
